# 🎬 فیلم‌باز (FilmBaz)

فیلم‌باز یک پلتفرم مدیریت و نمایش فیلم و سریال است که با Django توسعه داده شده و با هدف ارائه یک زیرساخت مقیاس‌پذیر، ماژولار و آماده استقرار در محیط Production طراحی شده است.

این پروژه علاوه بر مدیریت فیلم‌ها و کاربران، امکاناتی مانند جستجوی هوشمند، کشینگ، پردازش وظایف پس‌زمینه، ارتباط بلادرنگ (Real-Time) و سیستم پشتیبانی آنلاین را ارائه می‌دهد.

---

# ✨ قابلیت‌های اصلی

## مدیریت کاربران

* مدل کاربری اختصاصی
* ثبت‌نام و ورود کاربران
* ویرایش پروفایل
* آپلود تصویر پروفایل
* تغییر و بازیابی رمز عبور
* داشبورد کاربری
* لیست فیلم‌های ذخیره‌شده

## مدیریت فیلم و سریال

* ثبت و مدیریت فیلم‌ها
* مدیریت ژانرها
* نمایش اطلاعات کامل فیلم
* پوستر و تصویر پس‌زمینه
* امتیازدهی
* نمایش تاریخ انتشار
* مدیریت کشور و مدت زمان فیلم

## تعامل کاربران

* ثبت نظرات
* ذخیره فیلم‌ها
* سیستم ارسال تیکت و بازخورد

## مدیریت عوامل تولید

* بازیگران
* کارگردانان
* نویسندگان
* تهیه‌کنندگان

## پشتیبانی آنلاین

* گفتگوی زنده با پشتیبانی
* ارتباط لحظه‌ای مبتنی بر WebSocket
* ذخیره تاریخچه گفتگوها
* نمایش وضعیت خوانده شدن پیام‌ها

# 🏗 معماری پروژه

پروژه به صورت ماژولار طراحی شده است:

```text
account  → مدیریت کاربران
film     → مدیریت فیلم‌ها
people   → مدیریت بازیگران و عوامل تولید
support  → پشتیبانی آنلاین
```

همچنین از معماری Hybrid استفاده می‌کند:

```text
درخواست‌های HTTP
        ↓
Gunicorn
        ↓
Django

ارتباطات WebSocket
        ↓
Daphne
        ↓
Channels
```

# 🔍 جستجوی هوشمند

برای جستجو از قابلیت pg_trgm در PostgreSQL استفاده شده است.

ویژگی‌ها:

* جستجوی فازی
* تشخیص خطاهای تایپی
* رتبه‌بندی نتایج
* جستجوی مشابهت

به عنوان مثال اگر کاربر عبارت:

Interstelar

را جستجو کند، سیستم می‌تواند فیلم:

Interstellar

را نیز پیدا کند.

# ⚡ بهینه‌سازی عملکرد

## کشینگ

با استفاده از Redis:

* کش لیست فیلم‌ها
* کش جزئیات فیلم
* کش نظرات

## بهینه‌سازی Queryها

* select_related()
* prefetch_related()

برای کاهش تعداد Queryهای دیتابیس.

## بهینه‌سازی تصاویر

تصاویر به صورت خودکار Resize و Crop می‌شوند:

* پوستر فیلم
* تصویر پس‌زمینه
* تصویر پروفایل کاربران
* تصاویر بازیگران
* تصاویر عوامل تولید
